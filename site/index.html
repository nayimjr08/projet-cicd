🐳 Conteneurisation

Image basée sur Nginx 1.27 Alpine, construite via un Dockerfile multi-étapes avec healthcheck intégré.

  • Image légère (~40 Mo)
  • Permissions sécurisées
  • Healthcheck automatique

⚙️ Automatisation

3 workflows GitHub Actions orchestrent la chaîne complète du commit à la production simulée.

  • CI automatique à chaque push
  • Publication GHCR sur main
  • Promotion manuelle contrôlée

🔒 Sécurité

Permissions minimales, aucun secret dans le code, utilisation exclusive du GITHUB_TOKEN.

  • Principe du moindre privilège
  • Traçabilité par digest
  • Rollback possible

📐 Orchestration

Docker Compose pour la coordination de services et la simulation de scaling.

  • Service web + service tester
  • Réseau dédié isolé
  • Scaling simulé documenté
